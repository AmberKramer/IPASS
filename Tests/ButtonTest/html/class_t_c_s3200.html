Detailed Description

TCS3200 color sensor.

This is an ADT that makes it easier to use the TCS3200 color sensor. The pins used are stored as pin_in and pin_out, defined by hwlib. The type of the photodiode to be used is stored as an integer.

Scan colors.

This function scans for all 4 types of photodiodes and saves the results in the scan results variables. The results are saved at 20% of the original scan.
